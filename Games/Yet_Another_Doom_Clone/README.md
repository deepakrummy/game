## **Description 📃**
- It's yet another doom clone.

## **How to play? 🕹️**
- Kill the enemies and get through all the levels. But be careful, don't forget the way back.
- Controls:
	- Move - `WASD` or `Arrow` Keys
	- Aim/Shoot: Mouse/Click
