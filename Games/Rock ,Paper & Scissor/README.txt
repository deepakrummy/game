# **Rock, Paper & Scissor Game**

---

<br>

## **Description 📃** 
- This is a basic Rock,Paper,Scissor game.It is build on basic web tech stacks such as html , css , js .

## **functionalities 🎮** 
- random option generation for computer's play.
- user friendly interface

<br>

## **How to play? 🕹️**
- You have three buttons consisting of Rock ,Paper & Scissor. You have to click on one button.
- Based on user and computer's option, score is increased for respective winner.
