# **Retrohaunt** 

---

<br>

## **Description 📃**
- Someone or something is stealing our time. 
- Today was yesterday and tomorrow is nowhere to be seen. Detective Clay is on the case!

## **How to play? 🕹️**
- Controls:
	- Move - `W`, `A`, `S`, `D`
	- Jump - `Space`
	- Pause - `P`
	- Restart - `R`
