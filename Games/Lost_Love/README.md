# **Lost_Love** 

---

<br>

## **Description 📃**
- A maze game.
- You have to find the Love, but it's hiding in the shadows. It's against the clock.

## **How to play? 🕹️**
- Controls:
	- Move - W, A, S, D (or) Arrow Keys
	- Pause - P
