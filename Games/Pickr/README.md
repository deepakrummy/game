# **Pickr** 

---

<br>

## **Description 📃**
- A multi-gradient color picker game.


## **How to play? 🕹️**
- Goal of the game is to pick the closest color to the provided one in each of the 5 rounds. 
- The score will be provided as an error rate, which means 'the lower the better'.
