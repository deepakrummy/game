# **Teleport** 

---

<br>

## **Description 📃**
- A game about navigating spaces by teleporting
- Works on desktop or VR devices

## **How to play? 🕹️**
- Controls:
	- Move - WASD or Arrow Keys
	- Tap the screen with cursor to change your POV
	- Pause - P
