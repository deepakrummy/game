# **Jump_Off_Line** 

---

<br>

## **Description 📃**
- Jumping will erase the line you are running on.


## **How to play? 🕹️**
- Keep jumping to avoid enemies and stay on screen.
- See how many jumps you can last!
- Jump: click, tap, z or space.
